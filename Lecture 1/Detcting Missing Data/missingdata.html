Detecting Missing Data

Introduction

Missing data is a significant challenge in data analysis, affecting the quality and reliability of insights. Detecting missing data is the first step in understanding and addressing these gaps. This page covers the techniques to summarise, visualise, and understand missing data using Python.

Summarising Missing Data

Summarising missing data involves quantifying the number of missing entries in each column and calculating the percentage of missing values relative to the total dataset. 


# Calculate missing values
missing_summary = data.isnull().sum()
missing_percentage = (missing_summary / len(data)) * 100

# Combine into a summary table
missing_summary_table = pd.DataFrame({
    "Missing Count": missing_summary,
    "Missing Percentage": missing_percentage
})

The summary table provides a clear view of which columns have missing values and the extent of the issue, helping prioritise efforts to handle the missing data.

Visualising Missing Data

Visualisation techniques provide a quick and intuitive way to locate and understand patterns in missing data. The following methods are commonly used:

  • Heatmaps: Highlights missing data with a colour map for easy identification.
  • Bar Charts: Visualise the count and percentage of missing data per column.

Code Examples

Loading and Summarising Missing Data


# Load the dataset
data = pd.read_excel("Large_Business_Dataset_Missing_Values.xlsx")

# Summarise missing data
missing_summary = data.isnull().sum()
missing_percentage = (missing_summary / len(data)) * 100

# Combine into a summary table
missing_summary_table = pd.DataFrame({
    "Missing Count": missing_summary,
    "Missing Percentage": missing_percentage
})

# Display summary
print(missing_summary_table)

Heatmap for Missing Data


# Visualise missing data with a heatmap
sns.heatmap(data.isnull(), cbar=False, cmap="viridis")
plt.title("Heatmap of Missing Data")
plt.show()

Bar Charts for Missing Data

Visualising missing data as bar charts helps identify which columns require attention:


# Bar chart: Count of missing values
missing_summary.plot(kind="bar", color="steelblue")
plt.title("Count of Missing Values per Column")
plt.ylabel("Count")
plt.show()

# Bar chart: Percentage of missing values
missing_percentage.plot(kind="bar", color="darkorange")
plt.title("Percentage of Missing Values per Column")
plt.ylabel("Percentage (%)")
plt.show()

Conclusion

Detecting missing data is a crucial step in the data cleaning process. By summarising and visualising missing values, analysts can develop targeted strategies for imputation or removal, ensuring the integrity of their analyses.
