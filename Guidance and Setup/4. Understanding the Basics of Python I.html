Python Basics with VS Code

Introduction

Python is a versatile programming language that’s widely used for data analysis, automation, and web development. This guide introduces Python basics, including how to use pip for managing libraries, and integrates with Visual Studio Code (VS Code) for coding.

Step 1: Writing Python Code in VS Code

  1. Open VS Code and create a new Python file:
    • Click on File > New File.
    • Save the file with a .py extension (e.g., basics.py).
  2. Write your first Python program: 
    print("Welcome to Python Basics!")
  3. Run your Python code:
    • Right-click in the editor and select Run Python File.
    • Or use the terminal: type python basics.py and press Enter.

Step 2: Using Libraries in Python

Python libraries are pre-written code modules that provide additional functionality. Popular libraries include:

  • pandas: For data manipulation and analysis.
  • numpy: For numerical computations.
  • matplotlib: For data visualisation.
  • scikit-learn: For machine learning.

Libraries are managed using pip, Python’s package manager.

Step 3: Installing Libraries with pip

  1. Open the integrated terminal in VS Code (Ctrl + `).
  2. Install a library using pip. For example: 
    pip install pandas
  3. Check installed libraries by typing: 
    pip list
  4. Use the library in your Python script. For example: 
    
import pandas as pd
print(pd.__version__)
