R Basics with RStudio

Introduction

R is a powerful programming language used for statistical computing and data visualisation. This guide introduces the basics of R, including working with RStudio and managing libraries (packages) for data analysis.

Step 1: Writing R Code in RStudio

  1. Open RStudio and create a new R script:
    • Click on File > New File > R Script.
  2. Write your first R program: 
    print("Welcome to R Basics!")
  3. Run your R code:
    • Highlight the code and press Ctrl + Enter (Windows) or Cmd + Enter (Mac).
    • The output will appear in the Console panel.

Step 2: Using Libraries in R

R libraries (packages) extend the functionality of the language. Popular libraries include:

  • dplyr: For data manipulation.
  • ggplot2: For data visualisation.
  • tidyr: For data cleaning and reshaping.
  • shiny: For creating interactive web applications.

Libraries are installed and managed through the built-in package manager or via commands in the Console.

Step 3: Installing and Loading Libraries

  1. Install a library using the install.packages() function. For example: 
    install.packages("ggplot2")
  2. Load the library into your script: 
    library(ggplot2)
  3. Check installed libraries by clicking on the Packages tab in RStudio.
  4. Use the library in your code. For example: 
    
data(mpg, package = "ggplot2")
ggplot(data = mpg, aes(x = displ, y = hwy)) + geom_point()
