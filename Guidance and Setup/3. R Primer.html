Getting Started with R and RStudio

Introduction

This guide will help you set up R and RStudio, ensuring you’re ready to begin your data analysis journey. R is a powerful language for statistical computing, and RStudio provides an intuitive interface to make your workflow smoother.

By the end of this guide, you’ll have R and RStudio installed and ready to use.

Step 1: Install R

  1. Go to the official R website: https://cran.r-project.org/.
  2. Select your operating system (Windows, macOS, or Linux).
  3. Download and run the installer, following the on-screen instructions to complete the installation.
  4. Once installed, open a terminal or command prompt and type R --version to confirm R is installed successfully.

Step 2: Install RStudio

  1. Download RStudio from the official website: https://posit.co/download/rstudio-desktop/.
  2. Choose the version suitable for your operating system and run the installer.
  3. Follow the installation steps and launch RStudio once the setup is complete.

Step 3: Set Up Your Workspace

  1. Open RStudio and create a new R script:
    • Click on File > New File > R Script.
    • A new script editor will appear where you can write your R code.
  2. Save your script:
    • Click File > Save As....
    • Name your file with an .R extension (e.g., example.R).

Step 4: Run Your First R Program

  1. Type the following code into your R script: 
    print("Hello, Business Analytics!")
  2. Run the program:
    • Highlight the code and press Ctrl + Enter (Windows) or Cmd + Enter (Mac).
    • You will see the output [1] "Hello, Business Analytics!" in the Console panel.
