Getting Started with Python and Visual Studio Code

Introduction

This guide will help you set up Python and Visual Studio Code (VS Code) on your computer, ensuring you’re ready to dive into business analytics programming tasks.

By the end of this guide, you’ll have Python and VS Code installed, configured, and ready for use.

Step 1: Install Python

  1. Visit the official Python website: www.python.org.
  2. Click on the Downloads tab and choose the version suitable for your operating system (Windows, macOS, or Linux).
  3. Run the downloaded installer. Make sure to select the checkbox that says "Add Python to PATH" during installation.
  4. Once the installation is complete, open a terminal or command prompt and type python --version to verify the installation.

Step 2: Install Visual Studio Code

  1. Download Visual Studio Code from the official website: code.visualstudio.com.
  2. Run the installer and follow the instructions to install VS Code.
  3. After installation, open VS Code and install the Python extension:
    1. Click on the Extensions icon in the left-hand toolbar (it looks like four squares).
    2. Search for Python in the Extensions Marketplace.
    3. Click Install on the Python extension by Microsoft.

Step 3: Set Up Your Workspace

  1. Open VS Code and create a new folder for your project:
    • Click on File > Open Folder.
    • Select or create a folder where you want to save your files.
  2. Create a new Python file:
    • Click on File > New File.
    • Save it with a .py extension (e.g., example.py).

Step 4: Run Your First Python Program

  1. Type the following code into your Python file: 
    print("Hello, Business Analytics!")
  2. Run the program:
    • Right-click anywhere in the editor and select Run Python File.
    • Alternatively, use the integrated terminal at the bottom of VS Code and type python example.py.
  3. You should see Hello, Business Analytics! in the terminal output.
