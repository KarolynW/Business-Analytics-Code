Glossary of Terms

Business Analytics Concepts

Business Analytics
The process of using statistical methods, technologies, and software to analyse business data and make informed decisions.
Data-Driven Decision Making
The practice of making decisions based on data analysis rather than intuition or personal experience.
Key Performance Indicators (KPIs)
Metrics used to evaluate the success of an organisation or activity in achieving objectives.
Exploratory Data Analysis (EDA)
An approach to analysing data sets to summarise their main characteristics, often using visualisation techniques.
Predictive Analytics
The use of statistical models and machine learning techniques to predict future outcomes based on historical data.
Descriptive Analytics
The process of interpreting historical data to identify trends and patterns.
Prescriptive Analytics
The use of data and models to recommend specific courses of action.
Big Data
Large and complex data sets that traditional data processing applications cannot handle effectively.
Data Visualisation
The graphical representation of data to help people understand its significance.
Machine Learning
A subset of artificial intelligence that involves training algorithms to learn patterns from data and make predictions or decisions.
Data Cleaning
The process of preparing data for analysis by removing or correcting inaccurate, incomplete, or irrelevant data.
Statistical Modelling
A mathematical framework used to represent and analyse the relationships between variables in a dataset.

Python-Related Terms

Python
A high-level, interpreted programming language known for its simplicity and versatility.
Library
A collection of pre-written code that provides specific functionality, such as data manipulation or machine learning.
pandas
A Python library used for data manipulation and analysis, particularly with structured data.
numpy
A Python library for numerical computing, providing support for arrays and mathematical operations.
matplotlib
A Python library for creating static, interactive, and animated visualisations.
seaborn
A Python library built on matplotlib for making statistical graphics.
scikit-learn
A machine learning library in Python that provides simple and efficient tools for predictive data analysis.
pip
The package installer for Python, used to install and manage software packages written in Python.
IDE
Integrated Development Environment, such as VS Code, which provides tools for software development.
Jupyter Notebook
An open-source web application for creating and sharing documents that contain live code, equations, visualisations, and text.
Virtual Environment
An isolated Python environment that allows you to manage dependencies for a specific project.
API
An Application Programming Interface, which allows software applications to communicate with each other.
DataFrame
A 2-dimensional labelled data structure in pandas, similar to a table in a relational database.

R-Related Terms

R
A programming language and environment for statistical computing and graphics.
CRAN
The Comprehensive R Archive Network, a repository of R packages.
ggplot2
An R package used for data visualisation, implementing the grammar of graphics.
dplyr
A data manipulation package for R that provides a consistent set of verbs for data wrangling.
tidyverse
A collection of R packages designed for data science, including dplyr, ggplot2, and tidyr.
RStudio
An integrated development environment (IDE) for R.
shiny
An R package for building interactive web applications directly from R.
RMarkdown
A format for creating dynamic documents in R that combine code, output, and written analysis.
Base R
The default installation of R, which includes a variety of statistical and graphical functions.
Factor
A data structure in R used for categorical data.
Tibble
A modern version of a data frame in R, part of the tidyverse, designed for data science.

Key Concepts from the Lecture

Data Consistency
Ensuring data is uniform across systems, formats, and datasets to support accurate analysis and decision-making.
Data Transformation
The process of converting data from its original format into a format suitable for analysis, including scaling, encoding, and aggregation.
Scaling
A data preprocessing technique used to standardise the range of independent variables or features of data.
Data Imputation
The process of replacing missing or incomplete data with substituted values to maintain dataset integrity.
K-Nearest Neighbours (KNN) Imputation
A method of imputing missing values by using the average or most frequent values from the k-nearest data points.
Mean/Median Imputation
A simple imputation technique that replaces missing values with the mean or median of the column.
Multiple Imputation
An advanced statistical method that replaces missing data with multiple values to account for the uncertainty of imputed values.
Outlier Detection
The process of identifying data points that deviate significantly from the rest of the dataset, often indicating errors or unusual events.
Boxplots
A graphical representation of data that displays its distribution, quartiles, and potential outliers.
Isolation Forest
An algorithm used for anomaly detection by isolating outliers based on random partitioning of data.
Winsorisation
A statistical technique that limits extreme data values to reduce the effect of outliers on analysis.
Z-Score
A measure of how many standard deviations a data point is from the mean of its distribution.
Heatmaps
A data visualisation technique that uses colour gradients to represent the magnitude of data values.
Advanced Measures of Central Tendency
Statistical methods for determining the centre of a dataset, including weighted means and trimmed means.
Descriptive Statistics
Summary statistics that quantitatively describe or summarise features of a dataset, including measures of central tendency and dispersion.
Pair Plots
A type of visualisation that shows pairwise relationships between variables in a dataset, often used in exploratory data analysis.
Rug Plots
A visualisation technique that adds small tick marks along an axis to indicate data distribution.
Histogram
A graphical representation of the distribution of numerical data, where data is grouped into bins and represented as bars.
Linearity
A relationship between two variables where a change in one variable corresponds to a proportional change in the other.
Normal Distribution
A probability distribution that is symmetric about the mean, showing that data near the mean are more frequent in occurrence.
Data Cleaning
The process of identifying and correcting errors, inconsistencies, and inaccuracies in data to ensure its quality.
Trimmed Mean
A measure of central tendency that removes a percentage of the highest and lowest values before calculating the mean.
Weighted Mean
A measure of central tendency that assigns different weights to data points based on their importance or frequency.
Range
The difference between the maximum and minimum values in a dataset, providing a simple measure of dispersion.
Variance
A measure of how much values in a dataset differ from the mean.
Standard Deviation
The square root of the variance, representing the average distance of each data point from the mean.
