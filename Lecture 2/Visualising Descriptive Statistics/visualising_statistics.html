Visualising Descriptive Statistics

Introduction

Visualising descriptive statistics helps in interpreting datasets and identifying patterns or anomalies. This section demonstrates four common techniques for visualising data:

  • Heatmaps: Represent relationships between variables.
  • Histograms: Show frequency distributions of data.
  • Pair Plots: Explore relationships between multiple variables.
  • Rug Plots: Combine density curves with detailed data points.

Heatmaps

A heatmap provides a colour-coded representation of the correlation matrix, indicating the strength and direction of relationships between numeric variables. 


# Generate the correlation matrix
correlation_matrix = data.corr()

# Create a heatmap
sns.heatmap(
    correlation_matrix, 
    annot=True, 
    cmap="coolwarm", 
    vmin=-1, 
    vmax=1, 
    linewidths=0.5
)
plt.title("Correlation Heatmap")
plt.savefig("Correlation_Heatmap.png")
plt.close()

Visualisation Example

Histograms

Histograms depict the frequency distribution of a variable, with the option to overlay a density curve for smoother visualisation. 


# Generate histogram with density
sns.histplot(data["Age"], bins=10, kde=True, color="blue")
plt.title("Age Distribution")
plt.savefig("Age_Histogram.png")
plt.close()

Pair Plots

Pair plots display pairwise relationships between variables in a dataset, helping to identify correlations and clusters. Adding a hue allows categorisation based on a specific variable. 


# Generate pair plot
sns.pairplot(data, hue="Region", diag_kind="kde")
plt.savefig("Pair_Plot_With_Hue.png")
plt.close()

Rug Plots

Rug plots combine density curves with individual data points, offering a granular view of data distributions along an axis. 


# Create rug plot
sns.kdeplot(data["SpendingScore"], fill=True, alpha=0.6, color="blue")
sns.rugplot(data["SpendingScore"], color="red", height=0.1)
plt.title("Rug Plot and Density for Spending Score")
plt.savefig("SpendingScore_Rug_Plot.png")
plt.close()

Conclusion

Visualisation techniques provide powerful tools for interpreting descriptive statistics. By using heatmaps, histograms, pair plots, and rug plots, analysts can gain deeper insights into relationships, distributions, and trends within datasets.
